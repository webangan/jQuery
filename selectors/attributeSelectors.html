Has Attribute Selector [name]
$("[title]").css("background","#f00") 		Selects elements that have the specified attribute, with any value.
Attribute Equals Selector [name=”value”]
$("[title='abc']").css("background","#f00"); 		Selects elements that have the specified attribute with a value exactly equal to a certain value.
Attribute Not Equal Selector [name!=”value”]
$("[title!='abc']").css("background","#f00"); 		Select elements that either don't have the specified attribute, or do have the specified attribute but not with a certain value.
Multiple Attribute Selector [name=”value”][name2=”value2″]
$("[title*='abc'][align]").css("background","#f00") 		Matches elements that match all of the specified attribute filters ( As Many As attribute filter you can add )
Attribute Contains Selector [name*=”value”]
$("[title*='abc']").css("background","#f00"); 		Selects elements that have the specified attribute with a value containing a given substring
Attribute Ends With Selector [name$=”value”]
$("[title$='abc']").css("background","#f00"); 		elects elements that have the specified attribute with a value ending exactly with a given string. The comparison is case sensitive.
Attribute Starts With Selector [name^=”value”]
$("[title^='abc']").css("background","#f00"); 		Selects elements that have the specified attribute with a value beginning exactly with a given string.
Attribute Contains Word Selector [name~=”value”]
$("[title~='abc']").css("background","#f00"); 		Selects elements that have the specified attribute with a value containing a given word, delimited by spaces.
Attribute Contains Prefix Selector [name|=”value”]
$("[title|='abc']").css("background","#f00"); 		Selects elements that have the specified attribute with a value either equal to a given string or starting with that string followed by a hyphen (-).
